How to Upload Sounds to the Coin Box

  1. Log into the electronics-lab PC (Windows or Ubuntu).

  2. Open the Coin Box application. Under Windows, it may take a moment to start up…

  3. While the application is searching for the Coin Box, unplug and re-plug the device. Please do not insert any coins at this stage.

  4. Once the Coin Box appears, click Enter Config Mode. In this mode, the box ignores coins until it restarts.

  5. Pick an MP3 file (max. 5 seconds long) for each slot. The app shows the probability of each slot being played.

  6. If you ever want the defaults back, hit Factory reset samples.

  7. All done? Click Exit and restart Coin Box. The Coin Box will restart and start accepting coins again – now with your new sounds :)

Sounds auf die Coin-Box laden

  1. Melde dich am Feinwerkstatt-PC an (Windows oder Ubuntu).

  2. Starte das Coin-Box-Programm. Unter Windows kann es etwas dauern...

  3. Während die App sucht: kurz Stecker raus, wieder rein. Wichtig: Währenddessen keine Münzen einwerfen!

  4. Sobald die sie gefunden wurde, klicke Enter Config Mode. Im Config-Modus reagiert die Box nicht auf Münzen.

  5. Such dir eine MP3 (max. 5s lang) aus. Unter jedem Slot steht die Wahrscheinlichkeit für die Wiedergabe.

  6. Mit Factory reset samples holst du die ursprünglichen Sounds zurück.

  7. Fertig? Dann klicke Exit and restart Coin Box. Die Box startet neu und nimmt wieder Münzen an – mit deinen neuen Sounds :)
